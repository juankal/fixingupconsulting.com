## 📋 Páginas del Sitio

| Página | Descripción |
|--------|-------------|
| `index.html` | Homepage principal |
| `servicios.html` | Detalle de servicios de consultoría |
| `nosotros.html` | Historia, misión, visión, valores y equipo |
| `publicaciones.html` | Catálogo de libros publicados en OMP |
| `revista.html` | Revista Científica con artículos académicos |
| `contacto.html` | Información de contacto y formulario |

## 🛠️ Tecnologías

- **HTML5** semántico con atributos ARIA
- **CSS3** vanilla (sin frameworks) con variables CSS
- **JavaScript** vanilla — animaciones, partículas, AOS, counters
- **Google Fonts** — Inter + Outfit
- **SEO** — Schema.org LocalBusiness, Open Graph, geo meta tags

## 🎨 Características

- Diseño dark mode premium con colores navy + cyan
- Partículas animadas con red de nodos en el hero
- Animaciones scroll-triggered (AOS personalizado)
- Navbar glassmorphism con efecto scroll
- Contadores animados
- Catálogo de publicaciones OMP con filtros por categoría
- Selector de volúmenes para la revista científica
- Formularios funcionales con validación
- Botón WhatsApp directo
- 100% responsive (mobile, tablet, desktop)
- SEO avanzado con Schema.org, breadcrumbs y canonical URLs
